Getting Started¶

1. Get jiant¶

You can get jiant by cloning from GitHub:

git clone https://github.com/nyu-mll/jiant.git

2. Install dependencies¶

We recommend installing jiant’s dependencies in a virtual environment:

python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt

Note

If you plan to contribute to jiant, install additional dependencies with pip install -r requirements-dev.txt.

3. Use jiant¶

python jiant

Note

Running jiant without providing an experiment config argument (e.g., python jiant) will run a minimal example experiment. See Examples for examples that specify custom experiemental configuration via command line arguments or via configuration files.
